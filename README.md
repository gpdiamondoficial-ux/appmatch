# AppMatch

AppMatch é um MVP de plataforma de conexões baseado em compatibilidade de interesses,
onde os matches são sugeridos antes do contato direto.

## Stack
- Supabase (Auth + Postgres)
- Web App (MVP)
- SQL (estrutura inicial)

## Status
Fase 1 concluída:
- Banco de dados estruturado
- Autenticação preparada
- Segurança (RLS)
- Dados iniciais de interesses
